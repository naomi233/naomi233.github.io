## 系统快捷键

### 常用系统快捷键速查表

::: note
💡 **按键说明**：<br>

- `Cmd` = 命令键（⌘）<br>
- `Option` = 选项键（⌥）<br>
- `Ctrl` = 控制键（⌃）<br>
- `Shift` = Shift 键（⇧）<br>

:::

| 类别           | 快捷键                       | 功能说明                                      |
| -------------- | ---------------------------- | --------------------------------------------- |
| **窗口管理**   | `Cmd + M`                    | 最小化当前窗口                                |
|                | `Cmd + W`                    | 关闭当前窗口                                  |
|                | `Cmd + Q`                    | 退出当前应用程序                              |
|                | `Cmd + Option + W`           | 关闭当前应用的所有窗口                        |
|                | `Cmd + Option + M`           | 最小化当前应用的所有窗口                      |
|                | `Cmd + Control + F`          | 进入/退出全屏模式                             |
|                | `Cmd + Tab`                  | 切换应用程序（向前）                          |
|                | `Cmd + Shift + Tab`          | 切换应用程序（向后）                          |
|                | `Cmd + Option + →/←`         | 将窗口移动到右侧/左侧显示器                   |
| **桌面与空间** | `F3` 或 `Cmd + ↑`            | 打开 Mission Control（查看所有窗口和桌面）    |
|                | `Cmd + Shift + →/←`          | 切换到下一个/上一个桌面空间                   |
|                | `Cmd + Option + Shift + →/←` | 将当前窗口移动到下一个/上一个桌面空间         |
| **文本操作**   | `Cmd + C`                    | 复制选中内容                                  |
|                | `Cmd + V`                    | 粘贴内容                                      |
|                | `Cmd + X`                    | 剪切选中内容                                  |
|                | `Cmd + A`                    | 全选内容                                      |
|                | `Cmd + Z`                    | 撤销上一步操作                                |
|                | `Cmd + Shift + Z`            | 重做上一步操作                                |
|                | `Cmd + B`                    | 加粗选中文本（支持原生文本编辑器）            |
|                | `Cmd + I`                    | 斜体选中文本（支持原生文本编辑器）            |
|                | `Cmd + U`                    | 下划线选中文本（支持原生文本编辑器）          |
| **截图与录屏** | `Cmd + Shift + 3`            | 全屏截图                                      |
|                | `Cmd + Shift + 4`            | 区域截图（空格可切换窗口截图）                |
|                | `Cmd + Shift + 5`            | 专业截图/录屏工具（可选择区域、设置保存位置） |
| **系统功能**   | `Cmd + Space`                | 打开 Spotlight 搜索（文件/应用/信息）         |
|                | `Cmd + Control + Q`          | 立即锁定屏幕                                  |

## 终端快捷键速查表

| 类别           | 快捷键     | 功能说明                                         | 记忆提示 (Key)                               | 适用场景                                                           |
| :------------- | :--------- | :----------------------------------------------- | :------------------------------------------- | :----------------------------------------------------------------- |
| **光标移动**   | `Ctrl + F` | 光标**向前**移动一个字符                         | **F**orward                                  | 精细调整光标位置，替代右箭头键                                     |
|                | `Ctrl + B` | 光标**向后**移动一个字符                         | **B**ackward                                 | 精细调整光标位置，替代左箭头键                                     |
|                | `Ctrl + N` | 移动到**下一行**                                 | **N**ext line                                | 快速向下移动，替代下箭头键                                         |
|                | `Ctrl + P` | 移动到**上一行**                                 | **P**revious line                            | 快速向上移动，替代上箭头键                                         |
|                | `Ctrl + A` | 移动到当前行的**行首**                           | **A**head of line                            | 快速跳转到行首，特别适用于命令行                                   |
|                | `Ctrl + E` | 移动到当前行的**行尾**                           | **E**nd of line                              | 快速跳转到行尾，特别适用于命令行                                   |
| **文本编辑**   | `Ctrl + D` | **删除**光标下的一个字符（向前删除）             | **D**elete                                   | 当键盘没有独立的 `Delete` 键时特别有用                             |
|                | `Ctrl + H` | 删除光标前的一个字符（向后删除，等同 Backspace） | **H**（无特别含义，可联想“回退”）            | 替代 Backspace 键                                                  |
|                | `Ctrl + K` | 从光标处**删除到行尾**的所有内容                 | **K**ill                                     | 快速删除一行中光标后的内容，极为高效                               |
|                | `Ctrl + U` | 删除从光标处到**行首**的所有内容                 | 可联想为清空（Uproot）或撤销（Undo）行首内容 | 在终端中常用来快速清空已输入的命令                                 |
|                | `Ctrl + W` | 删除光标前的一个**单词**                         | **W**ipe word（或 **W**ord）                 | 快速删除光标前的一个完整单词                                       |
|                | `Ctrl + Y` | 粘贴（**拉回**）最近被删除的内容                 | **Y**ank                                     | 可以恢复被 `Ctrl+K`, `Ctrl+U` 等命令“杀死”的内容，像一个专用剪贴板 |
|                | `Ctrl + T` | **调换**光标前后两个字符的位置                   | **T**ranspose characters                     | 快速修正如将 “teh” 打成 “the” 这类错误                             |
| **历史与搜索** | `Ctrl + R` | **反向**递增搜索历史命令                         | **R**everse search                           | 在终端中，输入关键词即可动态搜索之前执行过的命令，效率极高         |
|                | `Ctrl + S` | **向前**递增搜索历史命令                         | **S**earch forward                           | 在搜索结果中向前查找                                               |
|                | `Ctrl + L` | 清屏，相当于执行 `clear` 命令                    | **L**ean (清晰)                              | 快速清除终端屏幕，保持界面整洁                                     |

::: tips
💡 进阶使用技巧

1. **适用场景广泛**：这套快捷键是 **Bash Shell 的默认设置**，在终端（Terminal 或 iTerm2）中功能最全。同时，它们也适用于 macOS 系统级的文本输入框（如备忘录、浏览器地址栏等）以及许多遵循 Cocoa 文本系统的应用 。
1. **优化键盘布局**：由于大量使用 `Ctrl` 键，你可以考虑在系统设置中将 `Ctrl` 键与使用频率较低但位置更顺手的 `Caps Lock` 键功能互换，这能显著提升操作的舒适度。
1. **组合使用**：你可以将移动和编辑快捷键组合起来。例如，使用 `Ctrl+A` 跳到行首，然后按 `Ctrl+K` 删除整行；或者使用 `Ctrl+R` 搜索历史命令后，用 `Ctrl+E` 跳到行尾进行修改。

:::

## 实用小技巧

1. **快速预览**：选中文件按 `空格键` 预览内容，无需打开应用
2. **强制退出应用**：`Option + Cmd + Esc` 打开强制退出窗口
3. **清空废纸篓**：`Shift + Cmd + Delete` 永久删除文件（跳过废纸篓）
4. **快捷键提示**：按住 `Cmd` 键可查看当前应用的所有快捷键
