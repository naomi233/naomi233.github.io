在没有公网 IP 的情况下，UGREENlink 可以访问到 NAS ，但只有门户系统提供的功能，无法直接访问 NAS 上的 WebDAV 服务。

这里介绍一种通过 UGREENlink 访问 NAS 上 WebDAV 的方法。

首先，确保你的 NAS 上已经启用了 WebDAV 服务，并记下 WebDAV 的访问地址。

然后，在 UGREENlink 的 Nginx 配置中添加以下内容：

添加文件 `/etc/nginx/conf.d/webdav.conf` ，内容如下：

```nginx
location = /dav {
    return 301 $scheme://$http_host/dav/;
}

location ^~ /dav/ {
    proxy_pass http://127.0.0.1:5005/;

    include /etc/nginx/proxy_params;

    proxy_pass_request_headers on;

    proxy_request_buffering off;
    proxy_buffering off;
    client_max_body_size 0;

    proxy_read_timeout 3600s;
}
```

重新加载 Nginx 配置：

```bash
sudo systemctl reload nginx
```

然后就可以通过访问 `https://xx.ug.link/dav` 来使用 WebDAV 服务，其中 `xx.ug.link` 是你在访问 `https://ug.link/你的UGREENid` 时获得的域名。
